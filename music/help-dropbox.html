Dropbox Help

Connect LizMusic to your own Dropbox music library

LizMusic can play music from your personal Dropbox storage after you authorize access with your own Dropbox account. LizMusic does not provide Dropbox accounts, hosted music, or shared libraries.

What you need

  • A Dropbox account that already contains your music files.
  • Supported audio files in a folder you can access from the signed-in account.
  • A browser on the Android device so Dropbox authorization can complete.

Typical setup flow

  1. Open LizMusic and go to Settings, then Storage, then Add Storage.
  2. Select Dropbox.
  3. Sign in with your own Dropbox account.
  4. Approve the requested access.
  5. Choose the folder path LizMusic should scan for playback.

Common Dropbox fixes

No music appears

Confirm the selected Dropbox folder contains supported audio files and that the same account owns or can access those files.

Authorization expired

Reconnect Dropbox in LizMusic if access was revoked, the token expired, or you changed Dropbox account settings.

File moved or renamed

Refresh the library if a file path changed in Dropbox after LizMusic indexed it.

Network is unstable

Use offline-friendly workflows for albums you need when mobile data or Wi-Fi is unreliable.

Need broader cloud playback help?

Compare Dropbox with Google Drive, OneDrive, WebDAV, local files, and offline cache if you are deciding where your music library should live.
