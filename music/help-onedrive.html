OneDrive Help

Connect LizMusic to your own OneDrive music library

LizMusic can play music from your personal OneDrive storage after you authorize access with your own Microsoft account. The app does not create or provide OneDrive accounts for users.

What you need

  • A Microsoft account with access to OneDrive.
  • Music files already stored in your own OneDrive folders.
  • A browser on your device so the Microsoft sign-in page can complete authorization.

Typical setup flow

  1. Open LizMusic and go to Settings, then Storage, then Add Storage.
  2. Select OneDrive.
  3. Sign in with your own Microsoft account.
  4. Approve the requested access.
  5. Choose the folder path you want LizMusic to use for music playback.

When playback stops working later

Authorization expired

Reconnect the OneDrive storage in LizMusic if Microsoft access was revoked or the session is no longer valid.

File moved or deleted

If the original file path changes in OneDrive, playback links may stop working until the library refreshes or the file is restored.

Wrong account signed in

Make sure the browser authorization flow uses the Microsoft account that actually holds the music files you want to play.

No files appear

Confirm your selected folder contains supported audio files and that you granted access to the right location.

Need account-side help?

For Microsoft account creation, password recovery, billing, or OneDrive service issues, follow Microsoft's own support and account recovery pages. LizMusic only handles the client-side playback connection.
