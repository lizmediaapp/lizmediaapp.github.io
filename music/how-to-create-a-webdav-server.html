WebDAV Setup

How to create a WebDAV server for personal audio files

A WebDAV server can be a NAS, a home server, or a hosted storage service. The exact interface depends on the provider, but the core setup is usually the same: create a folder, enable WebDAV, secure access, and test from Android.

Important copyright boundary

Only store and play files you own, created, recorded, or have permission to use. LizMusic does not provide music, karaoke tracks, backing tracks, shared libraries, download links, or copyright bypass tools.

Basic checklist

  1. Choose a NAS, home server, or hosting service that supports WebDAV.
  2. Create a private folder for your personal audio files.
  3. Enable WebDAV and copy the server URL.
  4. Use HTTPS if available and require a private username and password or app password.
  5. Connect from LizMusic and test browsing, playback, and offline-friendly workflows.
