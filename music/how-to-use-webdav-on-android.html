How to use WebDAV on Android

To use WebDAV on Android, you need a WebDAV server URL, your own login, and a folder that contains files you are allowed to access. LizMusic can then browse and play personal audio files from that storage source.

Legal personal audio only

Use WebDAV with audio files you own, created, recorded, or have permission to use. LizMusic does not provide content, shared libraries, download links, or copyright bypass tools.

Android setup checklist

  1. Confirm your WebDAV server works in a browser or file client.
  2. Copy the HTTPS URL if your provider supports it.
  3. Use your own username and password or an app-specific password.
  4. Choose the folder that contains your personal audio files.
  5. Open LizMusic and add WebDAV as a storage source.
