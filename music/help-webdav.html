WebDAV Help

Connect LizMusic to your own WebDAV or NAS music library

LizMusic can play music from WebDAV-compatible storage that you already manage. That might be a NAS, a home server, or another hosted WebDAV service under your control.

What you need

  • A WebDAV-compatible server that you own or administer.
  • The server URL, including the correct path and port if your setup uses a custom one.
  • A valid username and password or other credentials supported by that server.
  • A music folder on that server with files you are allowed to access.

Typical setup flow

  1. Enable WebDAV on your NAS or server first.
  2. Verify that the WebDAV URL works in another trusted client before adding it to LizMusic.
  3. In LizMusic, go to Settings, then Storage, then Add Storage.
  4. Select WebDAV and enter the server address, username, and password.
  5. Save the storage and test browsing or playback from a known folder.

Important notes

  • LizMusic does not set up WebDAV for you. It only connects to a server you provide.
  • If your server uses self-signed certificates or network restrictions, fix those on the server side first.
  • Keep the path and port consistent with the endpoint your NAS or server actually exposes.

Common problems

401 or 403 errors

Double-check your username, password, and permissions for the target folder. Also confirm you are using the correct WebDAV path.

Server not found

Make sure the URL is reachable from your phone and that the server is exposed to the network you are currently on.

Playback starts slowly

Check server performance, internet quality, and file size. Offline cache can help if your connection is unstable.

No music appears

Verify that the remote folder actually contains supported audio files and that your account can read them.

Need vendor-specific instructions?

If you are configuring a Synology, QNAP, Windows, or another third-party server, use that provider's official documentation for the server-side steps first, then return to LizMusic for the client-side connection.
